AI-Based Pedestrian Safety System

This project implements a real-time pedestrian detection system using computer vision techniques to enhance road safety by monitoring both driver and pedestrian environments.

Features
Real-time pedestrian detection using OpenCV
Object detection from live video streams and images
Driver-Side Module: Alerts drivers about nearby pedestrians
Crossing-Side Module: Monitors pedestrian movement for safe road crossing
Image preprocessing to improve detection accuracy
Technologies Used
Python
OpenCV
NumPy
How it Works
Captures real-time video input (camera/video feed)
Applies computer vision techniques to detect pedestrians
Draws bounding boxes around detected pedestrians
Triggers alerts based on proximity and movement
Applications
Driver assistance systems
Smart traffic management
Pedestrian safety monitoring
Accident prevention systems
