# рџ“Њ Project Overview

The **AI-Based Pedestrian Safety System** is a real-time computer vision application designed to improve road safety by detecting pedestrians and providing timely alerts to drivers and pedestrians.

The system uses **OpenCV** for pedestrian detection and **Flask** to provide a simple web-based interface for both the driver and pedestrian crossing modules.
